# RELEASE

## 公開準備の基本方針
- 「通常版公開」「カスタム版運用」「宣伝準備」を並行で進める
- 新機能追加は原則ストップし、公開品質の向上を優先する

## 公開直前チェック（P0）
- 致命的不具合がない
- 初回設定だけで使い始められる
- 対応機種説明が明確
- ストア説明文だけで価値が伝わる
- サポート連絡先がある
- カスタム版申込導線がある
- 料金と納品内容が明確
- プライバシー方針がある
- 公開告知文が準備済み
- 実機テスターから主要機種帯の確認結果が返ってきている
