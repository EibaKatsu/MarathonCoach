## T01 [P0] アプリ名を最終決定する

- 目的:

- 草案:
  - 正式名称: `レースナビ`。
  - 表記ルール: Store、LP、SNS、FAQで同一表記を使用。
  - アプリ内表示名: `AppName` を eng=`Race Navi` / jpn=`レースナビ` で統一する。

  - 注意点: 公開後の名称変更は検索性に影響するため原則行わない。
- 結果:
  - `アプリ名は「レースナビ」に固定し、申請素材・導線の表記を統一する。`

## T04 [P0] ストア説明文の詳細版を作る

- タスク内容:
  - ストア説明文の詳細版を作る
- 目的:
  - 導入前の不安を説明文だけで解消するため。
- 草案:
  - 詳細文構成: `対象ランナー` / `できること` / `設定方法` / `注意事項`。
  - 本文要点: 2設定で利用開始、距離別補給通知、心拍ペースに応じたアドバイス、個別分析はカスタム版で提供。
  - 詳細文(採用):
    - `レースナビ（通常版）`
    - `距離別の補給通知とペース調整アドバイスで、目標達成をサポート。`
    - `レース中の「このままのペースでいい？」を減らす、Garmin向け1画面データフィールドです。レース中の迷いは、判断に余計な集中力とカロリーを使ってしまう原因になります。目標タイム・距離・心拍ペースに合わせて次の行動を短く示し、走りに使うべきエネルギーを守りながら目標達成をサポートします。`
    - `主な機能: ペース調整アドバイス（ちょい上げ/そのまま/ちょい落とし）、ゴールタイム予測、距離別の補給通知、心拍/ドリフト表示、1画面表示。`
    - `通常版の補給仕様: フル=35分ごと、ハーフ=35分通知1回、10K/5K=通知なし。`
    - `設定は2項目: raceDistanceKm / targetTimeSec。`
    - `注意: 医療目的ではない。センサー状態により表示変動あり。目標達成を保証するものではない。`
  - 注意事項: 医療用途ではないこと、センサー誤差があることを明記。
  - 注意点: 詳細文は誇張を避け、公開時仕様と一致させる。
- 結果:
  - `詳細版は、心拍ペースに応じたアドバイスとレース中の迷い削減（集中力/カロリー消費抑制）を主軸にした文面で固定する。`

## T06 [P0] 設定方法の説明文を作る

- タスク内容:
  - 設定方法の説明文を作る
- 目的:
  - 初回利用で迷わず開始できる導線を作るため。
- 草案:
  - 設定導線（初心者向け）:
    - 1. Connect IQ Store で `レースナビ` をインストールし、時計と同期する。
    - 2. Connect IQアプリを開き、`マイデバイス` から `レースナビ` を選択する。
    - 3. `設定` を開いて入力項目を編集し、保存後に時計へ同期する。
  - 設定手順（通常版）:
    - `Race Distance (km)` を選択（42.195 / 21.0975 / 10 / 5）。
    - `Target Time Hour` と `Target Time Minute` を入力。
    - `Custom Mode Code` は通常版では空欄のままでOK（カスタム版利用時のみ入力）。
    - 保存して時計へ再同期後、ラン開始前にデータ画面で表示を確認する。
  - 補給通知の扱い: 通常版は距離別固定（フル=35分ごと、ハーフ=35分1回、10K/5K=通知なし）。
  - 案内文: `初めての方は「インストールして同期 → 設定を開く → 距離と目標タイムを入れる」の3ステップで開始できます。`
  - 注意点: 画面遷移を最小化し、1分以内に開始できる文量にする。
- 結果:
  - `設定方法説明は、Connect IQアプリの導線だけで設定画面に到達できる構成で固定する。`

## T07 [P0] スクリーンショットを4枚作る

- タスク内容:
  - スクリーンショットを4枚作る
- 目的:
  - ストア訪問者が利用シーンを即理解できる素材を揃えるため。
- 草案:
  - 推奨撮影機種（シミュレーター）: `epix2pro47mm`（見栄え重視の主機種）。
  - 予備機種: `fr265`（ランナー向け定番の見え方を確認したい場合のみ）。
  - 4枚構成: 1価値訴求 / 2走行中表示 / 3補給通知 / 4設定画面。
  - 撮影ガイド（4枚）:
    - 1枚目（価値訴求）: 通常時カード（`そのまま` など）で「迷いを減らす」印象が伝わる画面。
    - 2枚目（レース中表示）: ペース・心拍・予測タイムを1画面で訴求する画面。
    - 3枚目（補給通知）: `FUEL NOW` か補給警告状態が明確に見える画面。
    - 4枚目（設定画面）: `Race Distance (km)` / `Target Time Hour` / `Target Time Minute` が見える設定画面。
  - 撮影ルール:
    - 4枚とも同一機種・同一スケールで撮る（比較しやすくするため）。
    - シミュレーターのウィンドウ装飾は可能な範囲で除き、文字が読めるサイズで保存する。
  - 各画像に入れる短文: `迷わない表示` `補給忘れ防止` `2設定で開始` など。
  - 出力仕様: Store推奨サイズ、PNG、文字は画面幅の20%以上で可読性確保。
  - 注意点: 実画面と一致しないモック文言は使わない。
- 結果:
  - `4枚構成で確定。1枚目のみ日本語メイン、2〜4枚目は英語メインで作成。出力は shot01_value.jpg / shot02_one_screen.jpg / shot03_fuel_alerts.jpg / shot04_three_settings.jpg。`
