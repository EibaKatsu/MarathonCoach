## 実機テスター募集（X / Threads）
### 目的
- 対応機種の実機確認
- 初回設定の分かりにくさ確認
- 通知頻度/視認性/使い勝手の確認
- 不具合の早期発見

### 募集条件（例）
- 使用機種を共有できる
- ベータ版を試してフィードバックできる
- 10km以上のラン/レース想定で試せる
- Connect IQ導入に抵抗がない

### 回収項目（例）
- 使用機種
- インストール可否
- 初回設定で迷った点
- 見やすさ
- 通知の多さ
- 補給通知の使いやすさ
- 不具合内容
- 継続利用意向
