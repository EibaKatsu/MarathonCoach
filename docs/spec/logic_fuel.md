# logic_fuel

## 補給間隔
- 通常版（距離別固定）
  - フル（上記以外）: 35分ごとに通知
  - ハーフ（`20.8-21.4km`）: 35分通知を1回のみ
  - 10K（`9.5-10.5km`）: 通知なし
  - 5K（`4.5-5.5km`）: 通知なし
- カスタムモード: 30〜40分レンジで個別調整

## LAP操作による補給リセット
- LAP押下時点を補給完了として `fuelDueTimeSec` を再計算
- 遅れて補給した場合も、押下時点から再カウント
- `lapDebounceSec`（例: 20秒）以内の連続リセットは無効

## カード/メーター整合
- 残り `> 2:00`
  - メーター: 通常
  - カード: 通常表示（トグルなし）
- 残り `2:00〜0:01`
  - メーター: 注意
  - カード: `ACTION/FUEL` を3秒トグル
- 残り `<= 0:00`
  - メーター: 警告
  - カード: `FUEL NOW` 固定（最優先）

メーター警告文言:
- 日本語: `補給now / 補給後LAP`
- 英語: `Fuel now / Press LAP`

## 表示優先順位（固定ルール）
1. FUEL NOW（いま 補給）
2. HR超過（ちょい落とし）
3. DRIFT ON（水＋補給）
4. 通常（ACTION/FUELトグル）
