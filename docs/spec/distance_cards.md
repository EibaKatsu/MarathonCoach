# distance_cards

## フォーマット
3行固定フォーマット:
- 1行目: 距離ラベル（例 `8km` / `HALF` / `FULL`）
- 2行目: 意味づけ
- 3行目: 行動メッセージ

## 種目別ルール
- フル
  - 1kmごと通常スプリット
  - 21.1km: `HALF`
  - 42.2km: `FULL`
- ハーフ
  - 1kmごと通常スプリット
  - 10km: `10km`
  - 21.1km: `HALF`
- 10km
  - 1kmごと通常スプリット
  - 5km: `5km`
  - 10km: `10km`
- 5km
  - 1kmごと通常スプリット
  - 中間節目なし
  - 5km: `5km`

## 通常スプリット文言
- 序盤/中盤/後半の3距離帯テンプレートを使用
- 各距離帯3セットを `(km - 1) % 3` でローテーション

## 判定と表示
- 到達判定: `distance >= target - 0.02km`
- 表示時間: 7秒
- 表示優先順位:
  1. FUEL NOW
  2. HR OVER
  3. DRIFT ON
  4. 距離通知カード
  5. 通常カード（ACTION/FUEL）
- 警告カード表示中に到達した距離通知はスキップ
