# simulator_checklist

## シミュレーション確認観点（STEP13時点）
1. タイマー基準は `Data Field Timer Controls` の `Start/Pause/Lap` を使う
2. FUELが `35:00` から減る / Pause中停止 / Lapで `35:00` リセット
3. Lap連打時、20秒以内はリセットされない
4. 通常時カードはACTION固定、残り2分以内のみ3秒トグル
5. 期限超過後は `FUEL NOW` 固定（トグル停止）
6. HR/ZONE表示 `HR / Zn`、取得不可時 `HR / --`
7. HR超過が継続秒数で遷移し、回復5秒継続で解除
8. ゾーン取得不可時はHR超過カードに遷移しない
9. 基準区間は `20:00かつ3.0km` 後の連続10分で確定
10. DRIFT ON/OFF条件が機能する
11. `fr255 / fr57042mm / fr57047mm` でWatchdogクラッシュなし
12. `ja/en` 多言語表示と英語改行を確認
13. 距離通知カードが種目別ルールどおり（7秒、警告中到達スキップ）
