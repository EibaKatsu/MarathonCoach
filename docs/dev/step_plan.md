# step_plan

## 段階開発方針
- 一気に実装しない
- `1機能 + シミュレーター確認 + 1コミット` 単位で進める
- 問題が出たら次STEPへ進まず、そのSTEP内で修正・再確認する

## STEP一覧
1. 土台作成（Data Field最小構成、1行表示）
2. 設定2項目（レース距離/目標タイム）
3. 画面レイアウト枠（カード/HR/ZONE/FUEL/PACE/DIST/TIME）
4. ペース計算（10秒平均、`--:--`フォールバック）
5. 補給タイマー（35分固定）
6. LAP補給リセット + デバウンス
7. ACTION/FUELトグル（3秒）
8. 優先固定表示（FUEL overdue > HR超過 > DRIFT）
9. 心拍ゾーン上限 + HR超過継続判定
10. ドリフト検知（20–30分基準 + 10分窓）
11. 距離通知カード（1kmごと + 節目/ゴール、7秒表示）
12. 多言語対応（ja/en）
13. 最終調整（視認性/軽量化/安定性）
