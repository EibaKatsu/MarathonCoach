# TASKS

## 運用ルール
- 優先度は `P0 > P1 > P2`
- ステータスは `未着手 / 進行中 / 完了 / 保留`
- 担当は未定なら `自分`
- 期限は決まったら追記する
- 公開準備は「通常版公開」「カスタム版運用」「宣伝準備」を並行で進める
- 新機能追加は原則ストップし、公開に必要な品質向上を優先する
